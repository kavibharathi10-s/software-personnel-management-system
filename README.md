# 💼 Software Personnel Management System

## 📖 Project Overview

The **Software Personnel Management System** is a web-based application developed to simplify the management of software company operations. It provides separate modules for **Client, HR, Manager, and Employee**, allowing each user to perform their responsibilities efficiently through role-based access.

The system streamlines employee management, project allocation, attendance tracking, leave management, meeting scheduling, performance monitoring, and report generation within a single platform.

---

# 🎯 Objectives

- Simplify personnel management in a software organization.
- Reduce manual paperwork.
- Improve communication between clients, HR, managers, and employees.
- Track employee performance and project progress.
- Provide a centralized platform for managing organizational activities.

---

# ✨ Key Features

- Multi-role Login System
- Employee Registration
- Resume Upload
- Role-Based Access Control
- Client Project Management
- HR Employee Management
- Manager Task Allocation
- Leave Request Management
- Attendance Management
- Meeting Scheduler
- Project Tracking
- Performance Dashboard
- Performance Charts
- Report Generation
- Report Download
- Secure Data Management
